# 💳 AI Fraud Detection System

An AI-based machine learning project that detects fraudulent credit card transactions using classification algorithms.
The system analyzes transaction patterns and predicts whether a transaction is **fraudulent or legitimate**.

---

## 🚀 Features

* Detects fraudulent credit card transactions
* Uses Machine Learning models for prediction
* Handles highly imbalanced datasets
* Data preprocessing and feature scaling
* Model evaluation using accuracy, precision, and recall

---

## 🧠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## 📊 Dataset

The dataset used for this project is the **Credit Card Fraud Detection Dataset**.

Download it from Kaggle and place it in the `data` folder.

---

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/aaaayuuuushhh/AI_fraud_detection.git
```

Go to the project folder:

```
cd AI_fraud_detection
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the Project

Run the notebook or python script:

```
python main.py
```
